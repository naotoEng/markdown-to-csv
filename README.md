# markdown-to-csv README

## Features

Convert Markdown to CSV and the preview.

## How to Use

1. .mdファイルを開く。もしくは新規エディタにマークダウンのテキストを貼り付ける（言語の選択でMarkdownを選択する必要あり）
2. エディタを右クリックすると"Preview CSV", "Download CSV"のコマンドが表示されるので実行したい方をクリック
